
# 演示教程

本页介绍如何体验 MyUI 组件库的所有功能演示。

## 演示入口

所有演示页面位于 `src/Demos` 目录：
- `Demo.tsx`：主入口，集成导航和内容区。
- `NavBar.tsx`：导航栏，切换各组件演示。
- `ButtonsDemo.tsx`：按钮演示。
- `CardsDemo.tsx`：卡片演示。
- `PanelsDemo.tsx`：面板演示。

## 体验流程

1. 终端运行：
	```bash
	npm run dev
	```
2. 浏览器访问本地地址，进入演示主页。
3. 推荐体验顺序：
	- 按钮演示
	- 卡片演示
	- 面板演示

每个演示页面均有丰富示例和代码片段，适合边看边学。

如需自定义演示，可修改 `src/Demos` 下相关文件。
