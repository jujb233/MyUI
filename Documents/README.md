
# MyUI 组件库教程

> 本文档帮助你快速上手 MyUI，了解核心组件、配置方法与演示流程。

## 快速开始

1. 安装依赖：
	```bash
	npm install @jujb233/myui
	```
2. 引入组件：
	```tsx
	import { MyButton, MyCard, MyPanel } from 'myui';
	```
3. 查看演示：
	```bash
	npm run dev
	```
